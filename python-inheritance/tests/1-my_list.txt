This file contains doctests for the MyList class and its print_sorted method.

>>> MyList = __import__('1-my_list').MyList

Basic usage with append
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]

Already sorted list stays unchanged
>>> my_list = MyList([1, 2, 3])
>>> my_list.print_sorted()
[1, 2, 3]
>>> print(my_list)
[1, 2, 3]

Reverse-sorted list prints sorted
>>> my_list = MyList([5, 4, 3, 2, 1])
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[5, 4, 3, 2, 1]

Duplicates and negative numbers
>>> my_list = MyList([3, 1, -2, 3, 0])
>>> print(my_list)
[3, 1, -2, 3, 0]
>>> my_list.print_sorted()
[-2, 0, 1, 3, 3]
>>> print(my_list)
[3, 1, -2, 3, 0]

Empty list
>>> my_list = MyList()
>>> my_list.print_sorted()
[]
>>> print(my_list)
[]
