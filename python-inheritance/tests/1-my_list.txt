The ``MyList`` class prints a sorted version of itself without modifying the list.

>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]

It also works with negative numbers and duplicates.

>>> my_list = MyList([3, 1, -2, 3, 0])
>>> print(my_list)
[3, 1, -2, 3, 0]
>>> my_list.print_sorted()
[-2, 0, 1, 3, 3]
>>> print(my_list)
[3, 1, -2, 3, 0]
